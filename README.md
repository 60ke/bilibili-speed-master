# Bilibili Speed Master (哔哩哔哩倍速大师)

还在为 B 站原生最高 2 倍速感到太慢吗？想要更精准地控制视频节奏？「哔哩哔哩倍速大师」为你而来！这是一款专为 B 站用户设计的效率增强工具，助你突破播放限制，彻底掌控观影速度。

## 🚀 核心功能

- **超高倍速支持：** 提供 3x、8x、16x 预设选项，一键开启极速模式，追剧补番、学习网课效率翻倍。
- **精准微调：** 支持 0.1 - 16.0 范围内的任意倍速输入，精准到小数点后一位。
- **智能记忆功能：** 自动记录你最后的倍速设置，刷新页面或切换视频依然生效，无需重复操作。
- **双端控制：** 既可以在视频播放器的倍速菜单中操作，也可以直接点击浏览器扩展栏图标进行设定。
- **动态图标显示：** 浏览器图标实时显示当前播放倍速，状态一目了然。
- **丝滑快捷键：** 
  - `Shift` + `>` : 增加 0.5 倍速
  - `Shift` + `<` : 减少 0.5 倍速
  - `R` : 重置为 1.0 倍速

## 🛡️ 技术优势

- **底层级锁定：** 采用先进的 HTMLMediaElement 原型链拦截技术，能有效对抗播放器自带的自动重置逻辑。
- **极致轻量：** 极简 UI 设计，完美融入 B 站原生界面，无广告、无跟踪、无冗余权限。
- **完美适配：** 兼容 B 站新旧版播放器及各种视频页面。

## 📦 安装说明

1. 下载本仓库或 Release 中的 `bilibili-speed-master.zip`。
2. 在 Chrome 浏览器进入 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 将 ZIP 文件解压后，点击“加载已解压的扩展程序”选择解压后的目录即可。
